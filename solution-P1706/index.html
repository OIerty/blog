【洛谷】P1706【全排列问题】

首先全排列个数就是n!n!n!
接下来是C++在全排列里的过程：

next_permutation(a,a+n);//n表示全排列数，这可以让a[0]~a[n-1]按字典序进行下一全排列
prev_permutation(a,a+n);//进行上一去排列

不多说了 贴代码：

#include<bits/stdc++.h>
using namespace std;
int a[10],n,i,j=1,k;
int main(){
	scanf("%d",&n);
	for(i=0;i<n;i++){
		a[i]=n-i;//末项全排列，下一为第一个
		j*=i+1;//算总数n!
	}
	for(i=0;i<j;i++){
		next_permutation(a,a+n);
		for(k=0;k<n;k++)printf("%5d",a[k]);//做场宽
		printf("\n");
	}
	return 0;
}

就这么回事了
